Folder ini berisi texture planet yang akan digunakan oleh aplikasi AR Tata Surya.

File yang diharapkan:
- sun.jpg
- mercury.jpg
- venus.jpg
- earth.jpg
- mars.jpg
- jupiter.jpg
- saturn.jpg
- uranus.jpg
- neptune.jpg
- saturn-ring.png

Catatan:
- Aplikasi akan tetap berjalan tanpa file texture (menggunakan warna fallback)
- Format yang disarankan: JPG untuk planet, PNG untuk ring Saturnus
- Resolusi yang disarankan: 1024x512 atau 2048x1024 (equirectangular projection)
